# Agent Team Template

This directory is the reusable source template for creating per-project agent-team control planes.

## Template Home

Path: /Users/hay/Documents/agent-teams

## Default Project Copy Root

New per-project agent-team copies are created under:

```text
/Users/hay/Documents/agent-team-instances/
```

## Create A Project Team

```bash
./scripts/new-coding-project.sh /path/to/coding-project
```

By default this creates:

```text
/Users/hay/Documents/agent-team-instances/<project-name>-team
```
